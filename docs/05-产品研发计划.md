## 二、Phase 1 — Demo (第 1–4 周)

> **目标**: 证明技术可行性。在 1 个平台(macOS优先)实现截图采集→编码→保存最小闭环。

### 里程碑: 跑通第一个截图

| 周次 | 任务 | 产出 |
|------|------|------|
| W1 | 项目初始化：Cargo workspace、ci骨架、README | 项目骨架可编译 |
| W1 | 集成 xcap：monitor 枚举 + 区域截取 | 可以获取屏幕像素 |
| W1 | 集成 egui：空窗口运行、System Tray 图标 | GUI可启动 |
| W2 | 截图覆盖层(Overlay)：全屏透明+鼠标选区+坐标显示 | 可视化拖拽选区 |
| W2 | image编码输出：PNG保存到本地、复制到剪贴板 | 截图→文件/剪贴板 |
| W3 | 基础标注：矩形、箭头、文字(egui 渲染) | 截图上画标注 |
| W3 | 全局快捷键：Ctrl+Shift+A 触发截图 | 键盘触发截图 |
| W4 | 多显示器支持、边界处理、基础设置页 | 可用的演示版本 |
| W4 | **Demo Release**: 内部演示 + GitHub Release | v0.1.0 |

### Demo 验收标准
- [ ] 按快捷键弹起截图覆盖层
- [ ] 拖拽选中区域后截图
- [ ] 在截图上画矩形/箭头/文字
- [ ] 复制到剪贴板 / 保存PNG到桌面
- [ ] 系统托盘图标正常，内存<80MB

## 三、Phase 2 — MVP (第 5–12 周)

> **目标**: 截图全链路完整，基础录屏可录制。1个平台完整可用。

### Milestone 1: 截图功能完善 (W5–W6)

| 任务 | 说明 |
|------|------|
| 窗口截图 | 自动识别窗口、点击选中 |
| 延时截图 | 3/5/10秒倒计时 |
| 标注增强 | 椭圆、马赛克、序号、颜色选择器、撤销/重做 |
| 保存策略 | 自定义路径、命名规则、格式选择(PNG/JPEG/WebP) |
| 设置面板 | 快捷键自定义、常规设置 |

### Milestone 2: 录屏功能 (W7–W10)

| 任务 | 说明 |
|------|------|
| 区域录屏 | 拖拽区域→开始录制 |
| H.264 编码 | ffmpeg-next 软编码 |
| GPU 硬件编码 | VideoToolbox(macOS) / NVENC(Win) |
| 音频捕获 | 系统音频(macOS SCKit)、麦克风 |
| 录屏UI | 浮动控制条(录制时间/暂停/停止) |
| 基础GIF | 短时GIF录制 |

### Milestone 3: 打磨发布 (W11–W12)

| 任务 | 说明 |
|------|------|
| CI/CD | GitHub Actions 自动构建 |
| 打包 | macOS: .dmg; Win: .msi; Linux: AppImage |
| 文档 | 用户指南、快捷键表、FAQ |
| Bug修复 | 社区反馈修复 |
| **MVP Release** | v0.5.0 |

### MVP 验收标准
- [ ] 所有 M 类功能(M01-M10)全部实现
- [ ] 1个平台完整可用
- [ ] 截图+录屏核心流程无阻塞Bug
- [ ] CI/CD 自动构建测试通过
- [ ] 安装包可正常安装运行

## 四、Phase 3 — 1.0 (第 13–24 周)

> **目标**: 3 平台同时发布，贴图+PaddleOCR ONNX 等差异化功能到位。

### Milestone 1: 跨平台适配 (W13–W16)

| 任务 | 说明 |
|------|------|
| Windows DXGI 适配 | DXGI Desktop Duplication + WGC |
| Linux PipeWire 适配 | Wayland xdg-desktop-portal + X11 fallback |
| 平台差异处理 | 权限引导、文件路径、快捷键适配 |
| 3平台CI | 3个GitHub Runner并行构建 |

### Milestone 2: 贴图+PaddleOCR (W17–W20)

| 任务 | 说明 |
|------|------|
| 贴图系统 | 截图→贴到屏幕、缩放/旋转/透明度 |
| 贴图管理 | 多帖图、关闭、历史 |
| PaddleOCR集成 | ONNX Runtime 加载内嵌模型，中英文识别。离线可用 |
| 模型内嵌打包 | resources/models/ 随安装包分发，首次启动直接可用 |
| 模型自动更新 | 后台检查 registry 版本，增量下载 updated 模型 |

### Milestone 3: 发布准备 (W21–W24)

| 任务 | 说明 |
|------|------|
| 滚动截图 | 窗口内容滚动拼接长图 |
| 取色器 | 屏幕取色+多格式输出 |
| 分享功能 | 上传图床/生成链接 |
| 性能优化 | 冷启动<500ms、截图<80ms |
| i18n | 中文/英文/日文 |
| 官网 | 产品站 + 下载页 |
| **1.0 Release** | v1.0.0 |

## 六、风险与缓解

| 风险 | 概率 | 影响 | 缓解方案 |
|------|------|------|---------|
| xcap 在录屏侧能力不足 | 中 | 高 | 自研DXGI/SCKit补充，贡献上游PR |
| egui 文本渲染中文问题 | 高 | 中 | cosmic-text补充排版 |
| Linux Wayland 兼容性差 | 高 | 中 | X11 fallback，社区协作 |
| macOS 录屏权限弹窗频繁 | 中 | 低 | 权限持久化+引导流程 |
| 社区参与度低 | 中 | 高 | 详细CONTRIBUTING.md + Good First Issue |
