## 二、产品定位与背景

### 2.1 产品愿景
做全球第一款真正**跨平台、Rust原生、截图+录屏一体化**的开源工具。让 Windows/macOS/Linux 用户拥有一致的高速截图录屏体验。

### 2.2 背景与机会
1. **跨平台缺口**: 目前没有一款同时支持 Win/Mac/Linux 且体验一致的截图+录屏工具
2. **性能空白**: 市面上没有 Rust 原生的截图工具，Rust 的零成本抽象和内存安全是天然优势
3. **一体化趋势**: CleanShot X 已证明截图+录屏一体化是正确方向，但仅 macOS
4. **轻量录屏缺失**: OBS 太复杂，Loom 依赖云，缺一款开箱即用的本地轻量录屏

### 2.3 核心差异点 (vs 竞品)

| 对比维度 | 刹那 | ShareX | CleanShot X | Snipaste | Flameshot |
|---------|------|--------|-------------|----------|-----------|
| 跨平台 | ✅ Win/Mac/Linux | ❌ | ❌ | ✅ | ⚠️ |
| Rust原生 | ✅ | ❌ | ❌ | ❌ | ❌ |
| 截图+录屏一体 | ✅ | ✅ | ✅ | ❌ | ❌ |
| 贴图 | ✅ | ❌ | ✅ | ✅ | ❌ |
| 开源免费 | ✅ | ✅ | ❌ | ⚠️ | ✅ |
| GPU硬件编码 | ✅ | ✅ | ✅ | ❌ | ❌ |

## 三、用户画像

### 3.1 核心用户
| 用户类型 | 占比 | 核心场景 | 痛点 |
|---------|------|---------|------|
| **开发者** | 35% | 截图贴图对照文档写代码、Bug截图、录Bug复现 | 工具切换多、无跨平台一致性 |
| **设计师** | 25% | 设计稿对比、像素级取色、快速标注反馈 | 跨设备体验割裂 |
| **内容创作者** | 20% | 教程录制、长文截图、快速分享 | 截图+录制需多个工具 |
| **通用办公** | 20% | 文档截图、会议记录、快速分享 | 工具要么太复杂要么太弱 |

### 3.2 用户故事
- **开发者小王**: "我在公司用 Windows，回家用 Mac，想要同一套截图工具的快捷键和体验"
- **设计师 Lisa**: "截完图要立刻贴到草稿旁边对比、取色、标注。CleanShot 做到了但只在 Mac 上"
- **UP主老张**: "快速录一段教程 GIF 分享给团队，不想启动 OBS 那么重的工具"

## 四、功能需求 (KANO 分类)

### 4.1 M - 基本型需求 (Must-have)

| ID | 功能 | 描述 |
|----|------|------|
| M01 | 区域截图 | 鼠标拖拽选择区域，实时显示坐标和尺寸 |
| M02 | 窗口截图 | 自动识别窗口/控件边界，点击选中 |
| M03 | 全屏截图 | 截取全部显示器（含多屏） |
| M04 | 截图标注 | 矩形/椭圆/箭头/文字/马赛克/序号标注工具 |
| M05 | 复制到剪贴板 | 截图后自动复制，可直接粘贴 |
| M06 | 保存到本地 | 自定义路径、格式(PNG/JPEG/WebP)、命名规则 |
| M07 | 全局快捷键 | 可自定义的组合键，系统级热键注册 |
| M08 | 系统托盘 | 常驻托盘，右键菜单快速操作 |
| M09 | 区域录屏 | 选择区域录制视频(MP4/MKV) |
| M10 | 录屏含音频 | 系统音频 + 麦克风混音录制 |

### 4.2 O - 期望型需求 (One-dimensional)

| ID | 功能 | 描述 |
|----|------|------|
| O01 | 贴图(Pin) | 截图/剪贴板内容以浮动窗贴到屏幕最前，支持缩放/旋转/透明度 |
| O02 | 滚动长截图 | 窗口/网页内容自动滚动拼接为长图 |
| O03 | 延时截图 | 3/5/10秒倒计时截图 |
| O04 | OCR文字识别 | 截图后提取文字，PaddleOCR ONNX 本地引擎（中英文高精度），一键复制 |
| O05 | 取色器 | 屏幕任意位置取色，支持HEX/RGB/HSL多格式复制 |
| O06 | GIF录制 | 轻量 GIF 录制，可调节帧率和时长 |
| O07 | 标注历史 | 标注元素支持后期编辑（位置/颜色/大小） |
| O08 | 分享链接 | 上传到图床/云存储后生成分享链接 |
| O09 | 截图编号 | 按 #1 #2 #3 自动为截图添加序号标记 |
| O10 | 窗口录制 | 自动选择窗口并追踪录制 |

### 4.3 A - 兴奋型需求 (Attractive)

| ID | 功能 | 描述 |
|----|------|------|
| A01 | 自毁截图 | 截图仅在屏幕上停留，不落磁盘，阅后即焚 |
| A02 | 二维码识别 | 截图后自动识别二维码/条形码 |
| A03 | 屏幕尺子 | 像素级测量工具，支持水平和垂直测量 |
| A04 | 截图搜索 | 本地截图 OCR 索引（PaddleOCR ONNX），支持全文搜索历史截图 |
| A05 | 自动美化 | 截图自动添加圆角、阴影、背景色（预设模板） |
| A06 | AI抠图 | 截图主体智能抠图，去除背景 |
| A07 | 录屏自动编辑 | 录制后自动裁剪静默段、添加缩放动效 |
| A08 | 团队共享库 | 团队截图/录屏统一管理、搜索、标注协作 |

### 4.4 I - 无差异型 (暂不做)
- 直播推流
- 虚拟摄像头
- 视频会议集成

## 五、优先级 (MoSCoW)

### MVP (v0.5) — Demo 可运行
**M01-M08 + M09**（截图全链路 + 基础录屏）
- 区域/窗口/全屏截图
- 基础标注（矩形/箭头/文字/马赛克）
- 复制到剪贴板 + 本地保存
- 全局快捷键 + 系统托盘
- 基础区域录屏(无音频)

### v1.0 — 核心完整
**全部 M + O01-O07**
- 贴图功能
- 滚动长截图
- OCR 识别（PaddleOCR ONNX 本地引擎 + Tesseract fallback）
- 取色器
- GIF 录制
- 录屏含音频

### v1.5 — 进阶能力
**O08-O10 + A01-A05**
- 分享链接
- 自毁截图
- 屏幕尺子
- 二维码识别
- 截图搜索
- 自动美化

### v2.0 — 智能时代
**A06-A08**
- AI 抠图
- 录屏自动编辑
- 团队共享库

## 八、风险与应对

| 风险 | 级别 | 应对 |
|------|------|------|
| Linux Wayland 兼容性 | 高 | PipeWire + XDG Portal，社区共建 |
| macOS 录屏权限弹窗 | 中 | ScreenCaptureKit 标准权限，首次引导 |
| DXGI 多GPU环境 | 中 | 自动选择主显示器GPU，支持手动切换 |
| 社区维护力量不足 | 高 | 插件化架构降低贡献门槛，完善 CONTRIBUTING.md |
| egui 文本渲染局限 | 低 | 中文排版补充 cosmic-text 处理 |
